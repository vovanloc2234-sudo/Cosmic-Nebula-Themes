# Changelog

All notable changes to the "Cosmic Nebula" theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-03

### Added
- Initial release of Cosmic Nebula theme
- Main theme variant with full intensity cosmic colors
- Soft theme variant with muted colors for extended coding sessions
- Comprehensive color scheme covering all VS Code UI elements
- Full semantic highlighting support
- Support for all major programming languages:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS/SCSS
  - React/JSX
  - Vue.js
  - PHP
  - Java
  - C/C++
  - C#
  - Go
  - Rust
  - Ruby
  - JSON/YAML
  - Markdown
- Terminal color scheme with 16 ANSI colors
- Git decoration colors
- Bracket pair colorization support
- Optimized contrast ratios for accessibility
- Deep space aesthetic with purple nebula backgrounds
- Alien flora green accents for strings and success states
- Stellar blue for functions and information
- Nebula pink for constants and errors

### Features
- Eye-friendly color palette designed for long coding sessions
- Consistent visual hierarchy across all UI elements
- High contrast ratios meeting accessibility standards
- Immersive deep space coding environment
- Professional color scheme suitable for all development workflows

### Technical Details
- Complete VS Code color theme schema implementation
- Semantic token colors for enhanced syntax highlighting
- Comprehensive tokenColors rules for syntax elements
- Support for editor widgets and panels
- Git integration colors
- Diff editor highlighting
- Terminal integration
- Activity bar and sidebar theming
- Status bar and title bar styling
- Settings UI theming
